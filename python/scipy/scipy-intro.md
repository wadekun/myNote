# SciPy
Scipy基于Numpy。Numpy是N维数组容器，基于数组的纯数学计算库。     
Scipy是与具体领域有关的科学计算库。高等数学、信号处理、优化、统计和许多其它科学任务的语言扩展。     
比如做傅里叶变换，这是纯数学的，用Numpy，做滤波器，这属于信号处理模型了，在Scipy里找。
